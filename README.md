## 前端

### 安装依赖

```
yarn install
```

### 前端预览

```
npm start
```


### 开发默认启动

```
npm run dev
```

### 开发指定模块
```
npm run dev -- --define [module-name]
```

### 项目构建生成最新

```
npm run build
```



## 后端

1. 启动项目

```
yarn serve
```

2. 生成最新资源
```
npm run build
```

访问[这里](http://localhost:3001)
